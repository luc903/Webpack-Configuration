# Webpack Configuration

## Features

- Compresses Sass and css
- Compiles and transpiles Javascript
- Loads files to dist directory relative to the code
- Resolves for Vue.js

## How to get started
Once cloned, run the following to install all of the dependencies:

```
npm install
```

## How to use
For development, run:
```
npm run watch
``` 
For production, run:
```
npm run build
```
